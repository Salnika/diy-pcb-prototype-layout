## Démo (GitHub Pages)
Lien direct : https://salnika.github.io/diy-pcb-prototype-layout/

Le site est automatiquement déployé sur GitHub Pages à chaque push sur `master` via GitHub Actions.

## Dev
1. `bun install`
2. `bun run dev`

## Build
1. `bun run build`
2. `bun run preview`
